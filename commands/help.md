# コマンド一覧

~/.claude/commands/ フォルダ内のファイルを読み込み、以下のフォーマットで一覧表示してください。

## 表示フォーマット

```
## 使えるコマンド一覧

| コマンド | 用途 | 使うタイミング |
|---|---|---|
| /xxx | ... | ... |
```

各コマンドの「用途」と「使うタイミング」は、それぞれのコマンドファイルの内容から読み取って簡潔に1行で書いてください。

表の後に、以下の Claude Code 標準コマンドも別セクションとして表示してください：

## Claude Code 標準コマンド（設定不要で使えるもの）

| コマンド | 用途 | 使うタイミング |
|---|---|---|
| `/rewind` | 過去の操作を選んでファイル・会話・両方を巻き戻す | 変更を元に戻したいとき |
| `/compact` | 会話のコンテキストを圧縮してウィンドウを節約 | 会話が長くなってきたとき |
| `/clear` | 会話をリセットして新しいセッションを開始 | 別のタスクに切り替えるとき |

※ `/rewind` は `Esc` を2回押しても起動できます。

最後に「カスタムコマンドの詳細を知りたい場合は /コマンド名 と入力してください。」と案内してください。
