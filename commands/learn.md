# 学習の記録（instinct）

今のセッションで発見した重要な知見を instinct として保存してください。

## 手順

1. 今のセッションを振り返り、以下を抽出する：
   - うまくいったアプローチ・解決策
   - 試したがダメだったアプローチ（なぜダメだったか）
   - このプロジェクト固有の注意点・癖

2. 以下のフォーマットで `~/.claude/instincts/` に保存する：

ファイル名: `YYYY-MM-DD_[プロジェクト名].md`

```markdown
# Instinct: [タイトル]
日付: [今日の日付]
プロジェクト: [プロジェクト名]
信頼度: [高 / 中 / 低]

## うまくいったこと
- （具体的に）

## ダメだったこと
- （具体的に・理由も書く）

## 次回への注意点
- （次のセッションで最初から知っておくべきこと）
```

3. 保存後に「以下の instinct を保存しました」と内容を要約して報告する
