# Instinct 一覧確認

保存済みの instinct を一覧表示してください。

## 手順

1. `~/.claude/instincts/` フォルダ内のファイルをすべて読み込む
2. 以下の形式でまとめて表示する：

```
## 保存済み Instinct 一覧

### [プロジェクト名] — [日付]
- 信頼度: 高 / 中 / 低
- ポイント: （要約を1〜2行で）

...
```

3. 「このプロジェクトに関連する instinct を詳しく見たい場合は教えてください」と案内する

instinct ファイルが存在しない場合は「まだ instinct が保存されていません。/learn で記録を始めてください。」と表示する
