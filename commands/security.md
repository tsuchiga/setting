# セキュリティスキャン

AgentShield を使ってコードのセキュリティスキャンを実行してください。

## 手順

1. 以下のコマンドを実行する：
```bash
npx ecc-agentshield scan
```

2. スキャン結果を日本語で報告する：
   - 検出された問題を **重大度（Critical / High / Medium / Low）** 順に整理する
   - 各問題について、該当ファイル・行番号・問題の内容・修正方法を示す

3. 自動修正可能な問題がある場合は確認を取る：
   - 「以下の問題は自動修正できます。修正しますか？」と聞く
   - 承認を得てから `npx ecc-agentshield scan --fix` を実行する

4. 問題がない場合は「セキュリティスキャン完了：問題なし」と報告する
